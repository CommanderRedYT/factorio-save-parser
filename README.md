# factorio-save-parser

A javascript save file parser for factorio save files

```shell
$ npm i factorio-save-parser

# or 

$ yarn add factorio-save-parser
```

A huge thanks goes out to the people who have done the research in the save file
format [here](https://forums.factorio.com/8568) and [here](https://github.com/circlesabound/factorio-file-parser).
